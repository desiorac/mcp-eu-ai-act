# Examples

## quick_scan.py

Minimal example - scan a project and check compliance in 10 lines.

```bash
python3 examples/quick_scan.py /path/to/your/project
```

## full_report.py

Complete example - scan at all risk levels and save a JSON report.

```bash
python3 examples/full_report.py /path/to/your/project
```

## Running examples

All examples should be run from the repository root:

```bash
cd mcp-eu-ai-act
python3 examples/quick_scan.py .
```
